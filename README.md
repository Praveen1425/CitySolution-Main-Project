# 🌆 CitySolution – Blockchain Based Smart City Issue Reporting System

CitySolution is a full-stack web application designed to help citizens report and track civic issues in a transparent and secure way.
The platform integrates **Blockchain, IPFS, and Web Technologies** to ensure trust, accountability, and data integrity while managing public complaints.

## 🚀 Project Overview

CitySolution provides a digital platform where users can:

* Report city issues with location and media
* Track issue status in real-time
* Maintain transparent records using Blockchain
* Store data securely using decentralized storage

The system combines modern web development with decentralized technologies to improve trust between citizens and authorities.

## ⚙️ Installation & Setup

### 1️⃣ Clone the Repository

```
git clone https://github.com/Praveen1425/CitySolution-Main-Project.git
cd CitySolution-Main-Project
```

---

### 2️⃣ Backend Setup

```
cd backend
npm install
npm start
```

---

### 3️⃣ Frontend Setup

```
cd frontend
npm install
npm start
```

---

## 🔐 Environment Variables

Create a `.env` file inside the backend folder and add your own credentials.

⚠️ **Important:**
Never upload private keys or service account files to GitHub.
